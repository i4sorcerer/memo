##什么是Dubbo
Apache Dubbo是一款高性能Java RPC框架。
	1.RPC框架（服务治理技术）


##应用的发展进程RoadMap
	1. 单一应用架构（ORM）
	2. 垂直应用架构（MVC）
	3. 分布式应用架构（RPC）
	4. 流动计算架构（SOA）

	
##Dubbo的架构简单说明
节点		角色说明
Provider	暴露服务的服务提供方
Consumer	调用远程服务的服务消费方
Registry	服务注册与发现的注册中心
Monitor		统计服务的调用次数和调用时间的监控中心
Container	服务运行容器
	支持的容器
	Spring Container
	Jetty Container
	Log4j Container
